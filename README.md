# MAGICA-Media-Pembelajaran-Menggambar-Grafik-Canggih
MAGICA (Media Grafik Canggih) adalah media pembelajaran visual-interaktif berbasis Desmos untuk membantu memahami grafik fungsi melalui analisis turunan. Media ini melatih kemampuan spasial mahasiswa dalam mengamati perubahan bentuk grafik, meliputi naik-turun, kecekungan, titik ekstrem, titik belok, dan asimtot secara akurat.
